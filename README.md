## デモ
https://chomas-lb.github.io/event-clock/

配信や・イベント進行で、演者やスタッフが残り時間を一目で把握するためのWeb時計です。
事前に登録した簡単なスケジュールの表示も可能です。
色やフォントもお好みで変更してください。

イベントの現場用に見やすく、CMなどもない物が欲しくて作りました。

携帯端末はブラウザ、横画面表示で使用してください。
